# AI Resource Directories
A curated list of generic directories and mega-repos useful for extending any AI coding assistant (Antigravity, Cursor, Claude Code, etc.).

- [ECC Tools Directory](https://ecc.tools/) - Tool directory for AI extensions.
- [Everything Claude Code Mega Repo](https://github.com/affaan-m/everything-claude-code) - Comprehensive collection of extensions.
- [Awesome MCP Servers](https://github.com/punkpeye/awesome-mcp-servers) - A curated list of Model Context Protocol (MCP) servers.
- [Awesome Claude Skills](https://github.com/BehiSecc/awesome-claude-skills) - A curated list of AI skills.
- [Skill Codex](https://github.com/skills-directory/skill-codex) - Searchable directory of reusable skills.
- [Visual Skills Browser](https://cmux-artemzhutov.netlify.app/skills.html) - Visual interface for finding skills.
- [50+ Best MCP Addons](https://claudefa.st/blog/tools/mcp-extensions/best-addons) - High-level overview of top MCPs.
- [MCP Setup Guide](https://mcpcat.io/guides/best-mcp-servers-for-claude-code/) - Guides for setting up the best MCPs.
