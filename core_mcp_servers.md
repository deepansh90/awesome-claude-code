# Core Model Context Protocol (MCP) Servers
List of helpful MCP servers to expand AI coding assistant capabilities (e.g. Antigravity, Cursor, Claude). Use these standard tools across workflows.

## Web & Browsing
- [Playwright MCP](https://github.com/anthropics/mcp-server-playwright): Automate browsers and perform web testing.
- [Firecrawl MCP](https://github.com/nicholasoxford/firecrawl-mcp): Scrape websites and convert them to markdown data.

## Databases & Auth
- [Supabase MCP](https://github.com/supabase-community/supabase-mcp): Add database and authentication workflows.
- [Postgres MCP](https://github.com/nicholasoxford/postgres-mcp): Query and manage PostgreSQL databases.

## Development & Ops
- [GitHub MCP](https://github.com/nicholasoxford/github-mcp): Manage PRs, code reviews, and issues.
- [Linear MCP](https://github.com/nicholasoxford/linear-mcp): Track issues and sync ticketing.
- [Sentry MCP](https://github.com/nicholasoxford/sentry-mcp): Track bugs and catch production errors.
- [Dokploy MCP](https://github.com/Dokploy/mcp): Deploy and manage full-stack apps.

## Context & Workspace
- [Context7](https://github.com/upstash/context7): Live library docs integration.
- [Notion MCP](https://github.com/nicholasoxford/notion-mcp): Access workspace and wiki docs.
- [Figma MCP](https://github.com/nicholasoxford/figma-mcp): Pull design specs automatically.
- [Slack MCP](https://github.com/nicholasoxford/slack-mcp): Send automated status messages.
