# Skill: High-Agency Frontend (Good Taste)
**Description**: Prevents generic UI/UX outputs and ensures the AI acts as a sophisticated frontend engineer with "good taste".

## Guidelines for Frontend Design
1. **Never use generic placeholders.** Use varied, realistic dummy data.
2. **Typography Matters:** Do not use default fonts. Rely on standard clean sans-serifs (e.g. Inter, Roboto) or whatever the design specifies. Ensure sufficient contrast.
3. **Spacing and Layout:** Use proper padding/margins. Avoid cramped user interfaces. Standardize spacing rules (e.g., using a base unit like `4px` or `8px`).
4. **Color Palettes:** Avoid aggressive solid primary colors (pure red `#ff0000`, pure green). Use carefully curated HSL scales, subtle backgrounds, and soft gradients.
5. **Modern Patterns:** Integrate micro-animations, tasteful hover states, dark mode support gracefully, and glassmorphism only where appropriate.
6. **Robust Implementation:** Ensure responsive design by default. Do not construct desktop-only mockups.
