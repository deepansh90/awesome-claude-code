# Skill: Recurring Tasks (Loop)
**Description**: Pattern and macro recordings for repetitive, recurring developer tasks.

## Instructions
1. **Identify Repetition**: Actively monitor for repeated actions (e.g., scaffolding similar components, running the same 3-step test command sequence).
2. **Record Macro**: When a recurring pattern is identified, suggest creating a macro or script for it to save the user time in the future.
3. **Generalize**: If asked to "loop" a task, accept an isolated chunk of target files and flawlessly execute the same sequence of edits/steps across all targets.
